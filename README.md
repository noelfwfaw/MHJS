# MaaHengJiScript

</div>


> **MaaFramework** 是基于图像识别技术、运用 [MAA](https://github.com/MaaAssistantArknights/MaaAssistantArknights) 开发经验去芜存菁、完全重写的新一代自动化黑盒测试框架。
> 低代码的同时仍拥有高扩展性，旨在打造一款丰富、领先、且实用的开源库，助力开发者轻松编写出更好的黑盒测试程序，并推广普及。

## 脚本介绍
> 基于Maa的风之痕迹PC端（非模拟器）脚本，哼唧的模拟器实在是太卡了，猎金猎不了一点
- 只有猎金功能
- 食用方式：先启动游戏，再启动脚本，勾选任务后，点击开始任务。
- 默认前台操作，操作方式为Seize，因为后台大部分操作方式都无效，其中适配原神的操作方式可以用，但是会不停的影响鼠标的正常使用，而且有延迟会出现点击无效的情况，所以前台挂机最好。
- 不做正经维护，有任何问题自己fork解决，反正Maa已经做到了零代码基础都随便写了，不是什么难事
- 无法处理游戏黑屏的问题（真草台班子）
